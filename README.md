## 💡 소개
- 사용자가 판매 상품을 직접 등록하고, 등록된 상품들을 조회해볼 수 있는 오픈마켓 앱입니다. 
- 상품 등록, 수정, 삭제 기능이 있습니다.
- 상품 등록 시 사진 첨부가 가능합니다.
- 다른 사용자가 등록한 상품을 상세페이지에서 조회할 수 있습니다.
- REST API 연동을 통해 구현됩니다.
